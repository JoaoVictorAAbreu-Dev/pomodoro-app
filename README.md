# Pomodoro PWA

Aplicação web baseada na técnica Pomodoro para gerenciamento de foco e produtividade. O projeto foi desenvolvido como Progressive Web App (PWA), permitindo instalação no dispositivo e funcionamento offline.

## Objetivo

Fornecer uma ferramenta leve, responsiva e funcional para:

- Gerenciar ciclos de foco (Pomodoro)
- Controlar intervalos curtos e longos
- Melhorar consistência de estudo e trabalho
- Disponibilizar uso offline via Service Worker

## Funcionalidades

- Temporizador configurável
- Alternância automática entre foco e pausa
- Interface responsiva (desktop e mobile)
- Manifest para instalação como aplicativo
- Service Worker para cache e funcionamento offline

## Como Executar Localmente

Clone o repositório:

```bash
git clone https://github.com/SEU_USUARIO/pomodoro-app.git
```

Acesse a pasta do projeto:

```bash
cd pomodoro-app
```

Execute utilizando Live Server ou qualquer servidor HTTP local.
